# Learn API Rest NET Core

This repository is a **hands-on learning project** focused on building **REST APIs using ASP.NET Core**.  
Its purpose is to understand and apply the core concepts required to develop robust, scalable, and maintainable backend services.

---

## Project Goals

- Learn how to build RESTful APIs with ASP.NET Core
- Apply backend development best practices
- Understand the full API request/response lifecycle
- Serve as a study reference and portfolio project

---

## Tech Stack

- ASP.NET Core
- C#
- .NET SDK
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI


---

## Main Features

- CRUD operations
- HTTP Verbs usage (GET, POST, PUT, DELETE)
- Data validation
- Error handling
- Automatic API documentation with Swagger

---

## How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/Ivancabreradev/Learn-API-Rest-NET-Core.git

2. Restore dependencies:
   ```bash
   dotnet restore

3. Run the application:
   ```bash
   dotnet run

4. Open Swagger in your browser:
   ```bash
   https://localhost:50001/swagger
